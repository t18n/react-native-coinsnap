# Coin Snap

## Getting started
- Clone the repo
- Install dependencies `yarn && yarn ios:pod:install`
- Terminal 1: `yarn start`
- Terminal 2: `yarn android` or `yarn ios` or `yarn ios --device your_device_name`

## Features
- Register
- Login
- View Coin/EUR list
- View Profile info
- Add coin to Favorite
- Show response with Native Toast
